The Python hillshade implementation was compared with the classic hillshade generated by the Relief Visualization Toolbox (RVT) in QGIS. The goal was to verify whether the Python implementation produces comparable results when using identical parameters.
Both visualizations were generated with:


          - Azimuth = 350°
          - Altitude angle = 45°

A noticeable difference appears on the shaded (darker) slopes. The RVT result applies additional filtering, which produces a smoother terrain representation and reduces very dark areas. In contrast, the Python implementation preserves stronger shadow contrast. Because of this, subtle terrain features may be less visible, which can be important for archaeological interpretation.

There is an example of an archaeological site in Slovenia known as Tonovcov grad (Figure 2). The archaeological remains have been well studied and interpreted in previous surveys. In both visualizations the same azimuth and altitude angles were used for the hillshade calculation.
